Acerca del proyecto

Radio TESI es un proyecto de transmisión por internet y difusión institucional para compartir eventos, avisos y contenido escolar.

Objetivo

Crear un sistema completo de radio por internet con un sitio web que permita transmitir, publicar flyers y comunicar información del instituto.

Alcance del avance

Interfaz web funcional con secciones definidas. El streaming y el panel administrador se integran en siguientes entregas.

Próximos pasos

  • Conectar el reproductor a una URL real de streaming.
  • Agregar administración para subir flyers y noticias.
  • Definir programación y secciones de contenido.
